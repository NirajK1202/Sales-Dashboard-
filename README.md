# Sales Dashboard - Power BI

## Project Overview
This Power BI Sales Dashboard provides comprehensive insights into sales performance, customer behavior, and business growth through interactive visualizations and KPI tracking.

## Features
- Sales Performance Analysis
- Revenue & Profit Tracking
- Regional Sales Insights
- Customer Segmentation
- Product Performance Analysis
- Interactive Filters & Slicers
- KPI Monitoring Dashboard

## Tools Used
- Power BI
- DAX
- Power Query
- Data Modeling
- Excel/CSV Data Sources

## Key Insights
- Identified top-performing products and regions
- Analyzed sales trends and revenue growth
- Monitored business KPIs for decision-making
- Improved reporting efficiency through interactive dashboards
